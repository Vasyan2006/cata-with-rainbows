Основные моды:

Hairs - добавляет декоративные цвета для волос, глаз и кожи, которые можно выбрать при создании персонажа или изменить мутагенами после. Для графонистого отображение необходим тайлсет от Анона.
Более свежая версия мода тут: https://drive.google.com/drive/folders/1Li33j-q-qlAwYPW7sa4aDu3QsXUa7v5q
!!!!ВНИМАНИЕ!!! При установке этого мода ОБЯЗАТЕЛЬНО включать мод NPC_Traits, иначе все нпс будут лысыми !!!СПАСИБО ЗА ВНИМАНИЕ!!!
Мутанты из мода Mutant_NPCs так-же автоматически получают разноцветные глаза-кожу-волосы.




Прочее:

Avitaminosis - вырезанные из пк-ребаланса негативные эффекты от нехватки витаминов, обморожения конечностей и прочих бяк.

Seeds - позволяет крафтить семена пшеницы из зёрен и добавляет рецепты крафта других семян из плодов.

Shiny Micro - микроволновку можно заряжать батарейками и использовать для разогрева еды. Микроволновку, электроплитку и кофеварку можно устанавливать в автомобиль (но по функционалу они все становятся как электроплитка). 

Duffel Bag - позволяет прицепить баул к автомобилю для хранения лута.

Butthurt Dissipation CBM - добавляет бионику "Остужатель Пердаков КБМ".




Не рекомендуется к использованию:

Hornies - Васян игрался с lua. Позволяет состригать шерсть и спиливать рога и когти, которые затем отрастают обратно. Качество мода принципиально никем не тестировалось, претензии о работоспособности не принимаются.
